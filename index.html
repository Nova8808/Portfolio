Job Application Tracking Web App

A job search is a full time endevor and keeping oraganized and tracking progress can be useful. Using Visual Studio I created a ASP.NET MVC using Entity Framework. This allowed me to connect an SQL database to a basic CRUD web page. A job and followup class was created and corresponding database tables. Adding a controller for each allowed API style Create/Read/Update/Delete. I added an API endpoint so using a tool like Postman I am able to access the data though an API interface. Using this I know who I applied to, when I applied, the job title and the job location. On the follow ups page I am able to track and detail any application follow ups.
